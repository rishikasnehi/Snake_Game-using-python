# Snake Game using Python

A classic Snake Game implemented in Python. The objective is to control the snake to eat food, grow longer, and avoid colliding with the walls or itself.

## Features

- Pure Python implementation
- Incremental difficulty as the snake grows
- Score tracking

## Requirements

- Python 3.x

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/rishikasnehi/Snake_Game-using-python.git
   cd Snake_Game-using-python
   ```

2. Run the game:

   ```bash
   python snake.py
   ```
